## MIGRATION PROCESS

**Standard migration workflow:**
1. Export knowledge from source SIMA
2. Review export structure
3. Activate SIMA Import Mode
4. Load export file
5. Review for duplicates
6. Integrate into blank SIMA
7. Update indexes
8. Verify cross-references
