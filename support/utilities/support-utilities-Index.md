## USAGE GUIDELINES

**When to Use:**
- Upgrading SIMA version
- Moving to new installation
- Consolidating multiple SIMAs
- Major restructuring

**Migration Types:**
- Major version (e.g., v3 → v4)
- Minor version (e.g., v4.1 → v4.2)
- Patch version (e.g., v4.2.1 → v4.2.2)

**Recommended Approach:**
1. Review migration utility
2. Choose strategy
3. Follow checklist
4. Test in non-production
5. Execute migration
6. Validate thoroughly
