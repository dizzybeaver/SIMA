## CRITICAL: DD-1 vs DD-2 vs CR-1

**DD-1: Dictionary Dispatch (Performance)**
- Function routing optimization
- Used in LEE interface files
- Trade-off: Memory for O(1) lookup speed
- Files: DD1-01, DD1-02, DD1-03, DD1-DEC-##, DD1-LESS-##

**DD-2: Dependency Disciplines (Architecture)**
- Layer dependency management
- Used in SUGA architecture
- Rule: Higher layers depend on lower only
- Files: DD2-01, DD2-02, DD2-03, DD2-DEC-##, DD2-LESS-##

**CR-1: Cache Registry (Consolidation)**
- Central function registry
- Used in gateway.py and wrappers
- Pattern: Single import for 100+ functions
- Files: CR1-01, CR1-02, CR1-03, CR1-DEC-##, CR1-LESS-##

**Never confuse these three!**
