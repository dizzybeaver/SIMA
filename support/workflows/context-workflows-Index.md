## ADDING WORKFLOWS

**To add a generic workflow:**

1. Create workflow file in this directory
2. Use format: `Workflow-##-Title.md`
3. Include:
   - Purpose
   - Prerequisites
   - Steps
   - Success criteria
   - Examples
4. Update this index
