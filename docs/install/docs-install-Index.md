## SETUP WORKFLOW

**Standard installation process:**
1. Extract or clone blank SIMA
2. Configure file server (fileserver.php)
3. Set up web server (nginx/Apache)
4. Edit File-Server-URLs.md
5. Test file server access
6. Verify directory structure
7. Test mode activation
8. Create first project (optional)
9. Add first knowledge entry (optional)

---

## SYSTEM REQUIREMENTS

**File Server:**
- PHP 7.4+
- Web server (nginx/Apache)
- Public HTTPS access
- Directory read permissions

**Client:**
- Claude.ai account
- Text editor
- Git (optional)
