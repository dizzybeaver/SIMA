## MODE QUICK MATRIX

| Mode | Phrase | Output | Use When |
|------|--------|--------|----------|
| **General** | "Please load context" | Answers | Learning, asking |
| **Project** | "Start Project Mode for [PROJECT]" | Code | Building features |
| **Debug** | "Start Debug Mode for [PROJECT]" | Fixes | Troubleshooting |
| **Learning** | "Start SIMA Learning Mode" | Entries | Documenting |
| **Maintenance** | "Start SIMA Maintenance Mode" | Updates | Cleaning |
| **New Project** | "Start New Project Mode: [NAME]" | Structure | Setup |

---

## DECISION TREE

```
Need code? → YES → Project Mode
           → NO ↓
           
Need fix? → YES → Debug Mode
          → NO ↓
          
Need documentation? → YES → Learning Mode
                    → NO ↓
                    
Need cleanup? → YES → Maintenance Mode
              → NO ↓
              
Need new project? → YES → New Project Mode
                  → NO ↓
                  
Just asking? → YES → General Mode
```

✅ "Start SIMA Maintenance Mode"

## SWITCHING MODES

### Same Session
```
Mode 1 → work → switch → Mode 2 → work
```

### Same Project
```
Project Mode → build → Debug Mode → fix
```

### Different Projects
```
Project Mode for X → Project Mode for Y
```

### Always Explicit
Must use exact activation phrase to switch.
