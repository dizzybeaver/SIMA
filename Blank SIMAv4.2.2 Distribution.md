## QUICK START

### Step 1: Extract
```bash
# Extract archive
tar -xzf blank-sima-v4.2.2.tar.gz
# Or
unzip blank-sima-v4.2.2.zip
```

### Step 2: Configure
1. Set up fileserver.example.com (or your file server)
2. Update File Server URLs.md with your URL
3. Upload to your AI assistant

### Step 3: Activate
```
Upload File Server URLs.md to your AI assistant
Say: "Please load context"
```

### Step 4: Start Using
- **Learn:** "Start SIMA Learning Mode"
- **Build:** "Start Project Mode for [PROJECT]"
- **Maintain:** "Start SIMA Maintenance Mode"

**See:** [QUICK-START.md](QUICK-START.md) for detailed instructions
